Modelos de caixas nas CSS

as caixas são um tipo de elemento presentes nas CSS, com elas é possível de criar 'espaços' entre as marcações definidas pelo HTML.

Existem tipos de caixas, as 'Box-Level' e também as 'Inline-Level'.

Os níveis de título h1, h2, h3... São um tipo de caixa box-level, assim também como a marcação 'p' do HTML5

Exemplo de caixa 'box-level' - É separado por conteúdos

Os Links são um tipo de caixa Inline-Level. Eu sou uma Inline-Level level e fico entre uma marcação no HTML. Isso significa que o elemento visível estará na mesma linha que a marcação, como o 'p'.

Cada tipo de box-level está destacado por uma borda
